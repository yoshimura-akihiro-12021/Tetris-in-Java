# Tetris-in-Java

Javaを使ってテトリスを作る。

## 目標

Javaを使ってテトリスを作る。
段階としては以下の通り
- [x] 画面の作成
- [x] 描画空間の作成
- [x] 時間経過によるオブジェクト(ミノ)の移動
- [x] グリッド空間の生成
- [ ] ミノの形を作成・管理
- [ ] キーボードによるミノの移動
- [ ] ミノ同士、ウィンドウの衝突判定
- [ ] 消去判定
- [ ] ゲームオーバー判定
- [ ] 保存ミノの作成と取り出し
- [ ] UI作成
 - [ ] スコア表示
 - [ ] どのミノを保存しているか
 - [ ] 5つ先のミノまで表示
- [ ] ミノのランダム生成
- [ ] メインメニュー→ゲーム画面→リザルトの画面遷移
